التثبيت

المتطلبات

  • Python 3.x
  • MySQL Server
  • mysql-connector-python package

الخطوات

تثبيت الحزمة المطلوبة

pip install mysql-connector-python

تكوين اتصال MySQL

تحديث بيانات اتصال القاعدة البيانية في db_connection.py

host: localhost
user: root
password: mahmoud
database: training_mm

تهيئة القاعدة

python db_connection.py

قاعدة البيانات

علاقات الجداول

students ← training_reports → organizations
training_reports ← evaluations → internal_supervisors
organizations ← external_supervisors

students

  • student_id (INT, Primary Key)
  • student_name (VARCHAR(100))
  • program (VARCHAR(100))
  • enrollment_year (INT, 2015-2025)
  • cgpa (DECIMAL(3,2), 0.00-4.00)
  • email (VARCHAR(100), Unique)
  • phone (VARCHAR(15))

organizations

  • organization_id (INT, Primary Key)
  • organization_name (VARCHAR(100), Unique)
  • location (VARCHAR(100))

internal_supervisors

  • supervisor_id (INT, Primary Key)
  • supervisor_name (VARCHAR(100))
  • department (VARCHAR(100))

external_supervisors

  • supervisor_id (INT, Primary Key)
  • supervisor_name (VARCHAR(100))
  • organization_id (INT, Foreign Key)

training_reports

  • report_id (INT, Primary Key)
  • student_id (INT, Foreign Key)
  • organization_id (INT, Foreign Key)
  • report_date (DATE, 2023 only)
  • report_text (VARCHAR(1000))

evaluations

  • evaluation_id (INT, Primary Key)
  • report_id (INT, Foreign Key)
  • supervisor_id (INT, Foreign Key)
  • evaluation_date (DATE, 2023 only)
  • evaluation_score (DECIMAL(5,2), 0-100)
  • evaluation_comments (VARCHAR(1000))

users

  • id (INT, Auto Increment, Primary Key)
  • username (VARCHAR(50), Unique)
  • password (VARCHAR(255))
  • full_name (VARCHAR(100))
  • email (VARCHAR(100), Unique)
  • role (ENUM: admin, trainer, trainee, supervisor)
  • created_at (TIMESTAMP)

النشر

إعداد قاعدة البيانات (Database Setup)

  1. تثبيت MySQL Server على الخادم (Install MySQL Server)
    • Download from MySQL Official Website
    • Follow installation instructions for your operating system
  2. تكوين قاعدة البيانات (Configure Database) 
    mysql -u root -p
CREATE DATABASE training_mm;
USE training_mm;

إعداد البيئة (Environment Setup)

  1. تثبيت Python (Install Python)
    • Download Python 3.x from Python.org
    • Make sure to check "Add Python to PATH" during installation
  2. إنشاء البيئة الافتراضية (Create Virtual Environment) 
    # Create virtual environment
python -m venv venv

# Activate virtual environment
# Windows
venv\Scripts\activate
# Linux/Mac
source venv/bin/activate
  3. تثبيت المتطلبات (Install Requirements) 
    pip install mysql-connector-python

تكوين التطبيق (Application Configuration)

  1. تعديل إعدادات قاعدة البيانات (Update Database Settings)

    Edit db_connection.py with your production credentials:

    connection = mysql.connector.connect(
    host='your_production_host',
    user='your_production_user',
    password='your_secure_password',
    database='training_mm'
)

تشغيل التطبيق (Run Application)

  1. تهيئة قاعدة البيانات (Initialize Database) 
    python db_connection.py
  2. بدء التطبيق (Start Application) 
    python admin_panel.py
نصائح للنشر (Deployment Tips)
  • Use strong passwords in production
  • Regularly backup your database
  • Keep Python and all packages updated
  • Monitor system resources and logs
  • Consider using a process manager like PM2 or Supervisor

ملاحظة أمنية

يرجى تغيير بيانات اتصال القاعدة البيانية الافتراضية قبل النشر إلى الإنتاج. لا تعرض بيانات اتصال القاعدة البيانية الحساسة في رمزك أو نظام نسخ المستودع.
